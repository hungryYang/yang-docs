# 路由内容

### 普通路由
`Next.js` 使用文件系统来定义路由。

如：

| 文件夹/文件 | 路由 |
| --- | --- |
| `app/page.tsx` | '/' |
| `app/about/page.tsx` | '/about' |

### [Dynamic routes 动态路由](./route/dynamic-routes.mdx)


| 语法          | 是否必需 | 示例匹配路径 | 不匹配路径 |
|---------------|----------|--------------|------------|
| `[param]`     | 是       | `/a`         | `/`        |
| `[...param]`  | 是       | `/a/b/c`     | `/`        |
| `[[...param]]`| 否       | `/a/b/c` 或 `/` | 无       |

### [路由组和私有文件夹 Route Groups and private folders](./route/groups-private.mdx)


| 写法 | 路由 |
| --- | --- |
| `(folder)/auth/page.tsx` | '/auth' |
| `(folder)/login/page.tsx` | '/login' |
| `_folder` | 不生成路由 |
